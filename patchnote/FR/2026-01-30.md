# Détails de projets, statistiques GitHub & suivi d’activité

Cette mise à jour apporte une vraie montée en gamme du dashboard : meilleure navigation, pages projets détaillées et analyse fine de l’activité GitHub.

Date : 28 janvier 2026

## Cartes projets → pages de détail

- Ajout d’un bouton **« Voir le détail du projet »** sur chaque carte projet.
- Ce bouton redirige vers une **page dédiée par projet**.
- Interface fluide, moderne, avec icônes et hiérarchie visuelle claire.
- Toutes les informations importantes du projet sont centralisées sur une seule page.

> Exemple réel : au lieu de cliquer entre le site, GitHub et des stats externes, tu as tout sous les yeux, au même endroit.

---

## Page Statistiques GitHub

Nouvelle page dédiée à l’analyse GitHub d’un projet.

### Sélecteur de dépôt

- Sélecteur de répertoire GitHub obligatoire.
- Impossible d’afficher les statistiques sans au moins un dépôt sélectionné (évite les états vides incohérents).

### Statistiques globales

- Nombre de commits
- Nombre de contributeurs
- Lignes ajoutées / supprimées
- Étoiles (stars)
- Forks
- Taille du dépôt
- Date de création

> Exemple réel : comme un résumé « santé du projet » avant d’entrer dans le détail.

### Langages & contributeurs

- Carte des **langages utilisés**.
- Carte des **profils contributeurs** avec informations clés.
- Vision rapide de qui contribue et sur quoi.

---

## Historique et analyse des commits

### Graphique d’activité

- Graphique des commits avec sélecteur de période :
  - 7 derniers jours
  - 30 derniers jours

### Recherche par hash de commit

- Recherche directe par hash de commit.
- Sélecteur de période avancé :
  - 7 jours
  - 30 jours
  - 6 mois
  - 12 mois

### Détail d’un commit sélectionné

Lorsqu’un commit est sélectionné :

- Message et description
- Nombre de lignes ajoutées / supprimées
- Date et heure
- Nombre de fichiers modifiés
- Bouton de redirection vers GitHub

> Exemple réel : tu cliques sur un commit comme sur une ligne d’historique bancaire et tu vois exactement ce qui s’est passé.
