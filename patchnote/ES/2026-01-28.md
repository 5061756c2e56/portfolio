# Detalles de proyectos, estadísticas de GitHub y seguimiento de actividad

Esta actualización supone una mejora real del portfolio: mejor navegación, páginas de proyecto detalladas y un análisis fino de la actividad de GitHub.

Fecha: 28 de enero de 2026

## Tarjetas de proyecto → páginas de detalle

- Se añade un botón **«Ver el detalle del proyecto»** en cada tarjeta de proyecto.
- Este botón redirige a una **página dedicada por proyecto**.
- Interfaz fluida y moderna, con iconos y una jerarquía visual clara.
- Toda la información importante del proyecto queda centralizada en una sola página.

> Ejemplo real: en lugar de ir saltando entre el sitio, GitHub y estadísticas externas, lo tienes todo delante, en el mismo lugar.

---

## Página de estadísticas de GitHub

Nueva página dedicada al análisis de GitHub de un proyecto.

### Selector de repositorio

- Selector de repositorio de GitHub obligatorio.
- Es imposible mostrar estadísticas sin al menos un repositorio seleccionado (evita estados vacíos incoherentes).

### Estadísticas globales

- Número de commits
- Número de contribuidores
- Líneas añadidas / eliminadas
- Estrellas (stars)
- Forks
- Tamaño del repositorio
- Fecha de creación

> Ejemplo real: como un resumen de «salud del proyecto» antes de entrar en el detalle.

### Lenguajes y contribuidores

- Tarjeta de **lenguajes utilizados**.
- Tarjeta de **perfiles de contribuidores** con información clave.
- Visión rápida de quién contribuye y en qué.

## Historial y análisis de commits

### Gráfico de actividad

- Gráfico de commits con selector de período:
  - últimos 7 días
  - últimos 30 días

### Búsqueda por hash de commit

- Búsqueda directa por hash de commit.
- Selector de período avanzado:
  - 7 días
  - 30 días
  - 6 meses
  - 12 meses

### Detalle de un commit seleccionado

Cuando se selecciona un commit:

- Mensaje y descripción
- Número de líneas añadidas / eliminadas
- Fecha y hora
- Número de archivos modificados
- Botón de redirección a GitHub

> Ejemplo real: haces clic en un commit como en una línea de extracto bancario y ves exactamente lo que pasó.
